# PIT（Point-in-Time）汚染除去 検討資料

## 1. プロジェクト概要

米国株テーマ型モメンタム戦略「PRISMATIC」の4戦略（PRISM, PRISM-R, PRISM-RQ, G2-MAX）のバックテストにおいて、テーマ構成メンバーシップのPIT汚染が存在する。この汚染を除去または軽減する方法をMECEに検討したい。

### 戦略の構造
- **データソース**: stock-themes.com（日本語の米国株テーマ分類サイト）が提供する207テーマ × 846銘柄の分類
- **シグナル生成**: テーマ内の自己除外リターンに対するOLS回帰でα/β/R²を推定 → αでランキング → Top銘柄を選定
- **リバランス**: 20営業日ごと
- **BT期間**: 2000年7月〜2026年4月（310ヶ月、25年間）

## 3. PIT汚染の構造（定量的事実）

### 3.1 汚染の3軸

| 軸 | 説明 | 現在の状態 |
|---|---|---|
| P(価格) | 上場廃止銘柄の価格含むか | ✅ Norgate PIT で解決済み |
| U(ユニバース) | 投資可能銘柄群のPIT性 | ✅ SP500/Russell PIT membership で解決済み |
| **M(テーマ構成)** | **テーマメンバーシップのPIT性** | **❌ 未解決 — frozen(2026-04-08)を全期間に遡及** |

### 3.2 M汚染の定量的影響

**銘柄存在率（frozen 846銘柄がその年に存在していた割合）:**

```
2000年: 49%  ← 半分の銘柄がまだIPOしていない
2005年: 57%
2010年: 64%
2015年: 73%
2020年: 85%
2025年: 99%
2026年: 100%
```

**上場廃止銘柄: frozen membership内に0銘柄**
→ テーマ内のloser銘柄が完全に除外されている（テーマ内survivorship bias）

**汚染メカニズム:**
1. 2000年に存在しなかった銘柄（51%）を「テーマ構成員」としてα推定に使用 → 先読みバイアス
2. テーマから脱落した過去の銘柄が一切含まれない → 勝者バイアス
3. OLS α推定で「テーマ内自己除外リターン」をリグレッサーに使用 → 構成が違えばα推定値が変わる
4. 集中度が高い戦略（G2-MAX: 6テーマ）ほど汚染の影響が増幅される

### 3.3 GICS代替可能性の定量データ

テーマ構成銘柄のGICS Sub-Industry最頻値との一致率（purity）:

```
100%一致（完全代替可能）: 96 / 207テーマ (46%)
≧80%一致（良好な代替）:  112 / 207テーマ (54%)
≧50%一致（弱い代替）:    185 / 207テーマ (89%)
＜50%一致（代替不能）:     22 / 207テーマ (11%)
```

代替不能テーマの例: reit(20%), dividend(22%), content-studios(25%), nuclear-traditional(25%)
→ 複数GICSにまたがるクロスセクターテーマは代替困難

## 4. PIT汚染除去方法のMECE分類

### カテゴリA: 過去のテーマ構成を直接取得する

**A1. Wayback Machine（web.archive.org）からstock-themes.comの過去スナップショットを復元**
- stock-themes.comのAPIレスポンスまたはHTMLからテーマ構成を抽出
- 実現可能性: サイトの開始時期と保存頻度による。比較的新しいサイトの場合2020年以前のデータは期待薄
- 除去度: 取得できた期間分は完全除去

**A2. stock-themes.com運営への直接問い合わせ**
- 過去のテーマ構成変更履歴のJSONデータを提供依頼
- 実現可能性: 低（対応保証なし）
- 除去度: 全期間の履歴が得られれば完全除去

**A3. Forward PIT蓄積（capture_snapshot.py、既に稼働中）**
- 日次でテーマ構成をアーカイブし、将来のBTでPIT-safeデータを使用
- 現在3日分（2026-04-10開始）。最低6ヶ月（12リバランス）蓄積後に初のPIT-free BT可能
- 除去度: 将来分は完全除去。過去分は解決しない

### カテゴリB: データなしで統計的に補正する

**B1. IPO日フィルタ**
- 各リバランス日時点で未IPOの銘柄をテーマ構成から除外
- Norgateのfirst_dateで即実装可能
- 除去度: 「存在しなかった銘柄の先読み」を完全除去。ただし「テーマに入っていたが今は脱落した銘柄」の復元は不可能
- 影響推定: 2000年で51%の銘柄が除外される → テーマ構成が大幅に変わる

**B2. 存在加重縮小（Age-weighted shrinkage）**
- IPO後の経過年数に応じてα推定へのウェイトを縮小
- 例: IPO後1年未満 → α寄与を50%に、3年未満 → 75%に
- 除去度: B1の柔らかい版。急激な構成変化を避けつつバイアスを軽減
- 実装容易

**B3. メンバーシップ摂動ブートストラップ**
- テーマ構成を同一GICS Sub-Industry内でランダムに20%入れ替えたBTを100回実行
- 結果の分散から汚染による不確実性の幅を推定
- 除去度: 汚染を除去するのではなく、汚染量の上限を統計的に見積もる
- 計算コスト高（100回BT）

**B4. BM5 PIT校正（クロスバリデーション）**
- BM5は完全PIT（SP500 PIT membership使用）。別途「BM5にfrozen SP500を適用した偽汚染版」を作成
- BM5_clean vs BM5_frozen の差分 = 「frozen membershipによる過大評価率」
- この比率をテーマBTに補正係数として適用
- 除去度: 間接的だが定量的な補正。テーマ固有の汚染構造とSP500の汚染構造が異なる点に注意

### カテゴリC: テーマ依存を構造的に回避する

**C1. GICS Sub-Industry代替（部分的PIT化）**
- GICS purity 100%の96テーマ → GICSサブ産業で置換
- Norgate全銘柄のGICS分類 + Russell 3000 PIT membership → 各日時点での「PIT-safeテーマ構成」を再構成
- 除去度: 96テーマ（46%）は完全PIT化。残り111テーマはfrozenのまま
- 注意: GICSセクター分類自体も過去に再編あり（例: META 2018年 Tech→Communication Services）。Norgateは現時点のGICSのみ提供

**C2. 完全GICS置換（テーマ層の完全排除）**
- テーマ概念を放棄し、Russell 3000 PIT × GICS Industry = 「PITインダストリー」として全面再構成
- 除去度: M汚染を完全にゼロにできる
- 代償: stock-themes.comのテーマ粒度（例: 「AI半導体コア」「光接続トランシーバ」等の細分類）を失う
- 先行研究: SHADOW_BOOKで「テーマ層はfactor/industry carrier → 消してはいけない」と結論

**C3. Individual stock α（de-theming）**
- テーマ層を通さず、個別銘柄のSPY対比αを直接計算
- 除去度: テーマ構成への依存をゼロに
- 先行研究: SHADOW_BOOKのbear_phase4で研究済み → 棄却（テーマ層がfactor carrierとして機能しており、除去するとα推定精度が低下）

### カテゴリD: 汚染の影響を測定し判断基準を変更する

**D1. 汚染量の定量推定**
- B1（IPOフィルタ）適用前後のBT差分で「最低限の汚染量」を測定
- 汚染がBT結論（A5>A4、制約の有効性等）を変えるかを判定
- これ自体は汚染を除去しないが、意思決定に必要な情報を提供

**D2. 絶対値禁止・相対比較のみ（Silver Layer規則）**
- 既にSHADOW_BOOKで運用中の規則:「A5-A4差分の方向のみ信頼、絶対CAGR/Sharpeは参考値」
- ただしA5のα推定自体がM汚染で歪むため、差分の大きさも過大評価の可能性

**D3. Forward-only判断**
- 過去BTを一切の判断材料にしない。capture_snapshot.pyの蓄積データのみで意思決定
- 除去度: 将来分は完全。過去に対する補正は行わない
- 実行可能時期: 最低6ヶ月後（2026年10月以降）
